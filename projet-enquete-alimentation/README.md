# Étude statistique sur les pratiques alimentaires des étudiants

## Présentation du projet

**Équipe :** 4 étudiants
**Durée :** 4 mois

Ce projet mobilisait les trois grandes compétences de notre formation en Science des Données : collecte, analyse et valorisation des données.

Nous avons mené une enquête complète sur les pratiques alimentaires des étudiants, depuis la conception du questionnaire jusqu'à l'analyse et la restitution des résultats. 
L'objectif était de comprendre les facteurs qui influencent les choix alimentaires des étudiants à travers une démarche statistique rigoureuse.

---

## Objectif de l'étude

Chaque groupe devait répondre à une sous-problématique spécifique à partir des données collectées.
`"Les arbitrages qui guident les pratiques alimentaires des étudiants et des personnels sur le campus de 
Villetaneuse sont-ils socialement déterminés (sur les plans économiques, écologiques, nutritionnels, 
organisationnels, culturels) ?"`

Notre mission consistait à :

* concevoir un questionnaire pertinent ;
* collecter des données fiables ;
* analyser les réponses obtenues ;
* valider statistiquement nos hypothèses ;
* présenter les résultats de manière claire et accessible.

---

## Technologies et outils utilisés

## Préparation et nettoyage des données

Après la phase de collecte, les données ont été nettoyées sous Excel afin de garantir leur qualité avant les analyses.

Cette étape comprenait :

* la vérification des réponses ;
* la correction des incohérences ;

* la préparation des données pour les analyses statistiques.

---

## Analyses statistiques réalisées

Afin de répondre à notre problématique, nous avons utilisé plusieurs méthodes statistiques :

* tris à plat ;
* croisements de variables ;
* statistiques descriptives ;
* tableaux de contingence ;
* tests du Khi-deux (χ²).

Ces analyses nous ont permis de vérifier statistiquement les hypothèses formulées en début d'étude et d'identifier les relations significatives entre différentes variables.
